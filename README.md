# E-Commerce Website

This is a simple E-Commerce website created as part of my Web Development Internship at CodSoft. The project was built to practice basic web development concepts and to understand how an online shopping interface works.

Live Website:  
https://rathimaan.github.io/codsoft/ecommerce/

## About the Project

The website allows users to browse different products and view their details in a simple and clean layout. The main goal of this project was to learn how to design a structured website and improve my front-end development skills.

## Features

- Product listing page  
- Clean and simple user interface  
- Responsive layout  
- Basic product display and navigation  

## Technologies Used

- HTML  
- CSS  
- JavaScript  

## Learning Outcome

While building this project, I practiced creating web page layouts, styling using CSS, and adding basic interactivity with JavaScript. It also helped me learn how to deploy a project using GitHub Pages.
